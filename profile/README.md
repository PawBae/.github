## What is PawBae? / PawBae 是什么？

PawBae is a desktop pet powered by AI that lives alongside your coding agents — watching them work, reacting in real time, and keeping you company through long coding sessions.

PawBae 是一只由 AI 驱动的桌面宠物，它陪伴在你的编程 Agent 旁边——实时观察它们的工作，做出反应，在漫长的编码时光中陪伴你。
